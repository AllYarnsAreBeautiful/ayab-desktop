[1.0.0]

Change in versioning scheme: From now on, the AYAB desktop software and firmware are using the Semantic Versioning 2.0.0 (https://semver.org/) versioning scheme. Pre-releases are marked with a suffix, e.g. 1.0.0-rc1, which stands for "release candidate 1"
